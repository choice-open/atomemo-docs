# 插件

使用自定义插件扩展 {{PRODUCT_NAME}} 以增强你的工作流并与第三方服务集成。

## 功能

插件系统允许你：

- **创建自定义节点类型** - 构建针对你特定需求的自定义节点
- **扩展工作流功能** - 为你的工作流添加新功能
- **集成第三方服务** - 无缝连接外部服务和 API
- **分享和发布插件** - 为社区插件市场贡献代码

## 可用资源

- **插件市场** - 浏览和安装社区开发的插件
- **插件开发** - 使用 SDK 创建自定义插件
- **插件管理** - 轻松安装、更新和配置插件
- **插件 API** - 完整的插件开发接口文档
