# Plugins

Extend {{PRODUCT_NAME}} with custom plugins to enhance your workflows and integrate with third-party services.

## Features

The plugin system allows you to:

- **Create custom node types** - Build custom nodes tailored to your specific needs
- **Extend workflow functionality** - Add new capabilities to your workflows
- **Integrate third-party services** - Connect external services and APIs seamlessly
- **Share and publish plugins** - Contribute to the community plugin marketplace

## Available Resources

- **Plugin Marketplace** - Browse and install community-developed plugins
- **Plugin Development** - Create custom plugins using the SDK
- **Plugin Management** - Install, update, and configure plugins easily
- **Plugin API** - Complete plugin development interface documentation
